# 数据分析助手

这是一个使用Streamlit构建的数据分析Web应用。

## 功能特点

- 支持CSV文件上传
- 数据预览
- 描述性统计分析
- 数据可视化（散点图和相关性热力图）
- 内置示例数据

## 在线使用

访问：[[Streamlit Cloud链接](https://f6wjuywwysgtvevvgfgyct.streamlit.app/)]

## 本地运行

1. 克隆仓库
2. 安装依赖：`pip install -r requirements.txt`
3. 运行应用：`streamlit run app.py`
